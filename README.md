<h2>Getting Started</h2>
* Download and install Python 3.10<br>
* Download and install Git.
<h4>Cloning</h4>
<code>git clone https://github.com/ahghanbari/poll_request_app.git</code><br>

<h4>Virtual Environment</h4>
<code>cd poll_request_app</code><br>
<code>pip install virtualenv</code><br>
<code>python3 -m venv .venv</code><br>
<code>source .venv/bin/activate</code><br>
<code>pip install -r requirements.txt</code><br>

<h4>Database Migration</h4>
<code>python manage.py makemigrations</code><br>
<code>python manage.py migrate</code>

<h4>For Admin Panel</h4>
<code>python manage.py createsuperuser</code>

<h4> To run the program in local server use the following command </h4>
<code>python manage.py runserver</code><br><br>

<p>Then go to <code>http://127.0.0.1:8000</code> in your browser</p>
